# PixelPalette

![PixelPalette Banner](https://via.placeholder.com/1000x200.png?text=PixelPalette)

## Overview

PixelPalette is a dynamic pixel art editor designed for both beginners and professionals. With a sleek interface and powerful tools, you can create stunning pixel art, animations, and share your creations with a growing community.

## Features

- Intuitive drawing tools including brush, eraser, fill, and color picker
- Layer management for complex pixel compositions
- Animation timeline with frame-by-frame editing
- Online gallery to share and explore pixel art from other users
- Export your art as PNG or GIF with customizable resolution

## Tech Stack

- **Frontend:** React, Redux, Canvas API
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JWT

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/jonesdcatherined/PixelPalette.git
   ```

2. Navigate into the project directory:

   ```bash
   cd PixelPalette
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

## Usage

- Open your browser and navigate to `http://localhost:3000`
- Create a new canvas or open an existing project
- Use the toolbar to draw, animate, and edit your pixel art
- Save your work locally or upload to the community gallery
